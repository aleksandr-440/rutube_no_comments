# Rutube No Comments

Расширение для Firefox, которое скрывает блоки комментариев на rutube.ru.

## Установка

**Firefox:** [Расширение доступно для установки на официальном сайте Mozilla](https://addons.mozilla.org/ru/firefox/addon/rutube-no-comments/)

**Chrome:** TODO

## Использование

Нажмите на иконку расширения и используйте переключатель для включения/выключения скрытия комментариев. Состояние сохраняется между сессиями.

Расширение не собирает и не передает данные пользователя.
